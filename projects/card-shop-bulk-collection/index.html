Dashboard October 2025

Card Shop Bulk Collection

A local Magic: The Gathering shop had thousands of cards priced at a flat 25 cents each, including some worth significantly more. I scanned the entire collection, enriched it with live market data, and built a Power BI dashboard so the owner could finally see what they actually had.

Python Power BI Scryfall API Manabox
Background

The Problem

As a regular at a local card shop, I was browsing the bulk section one day and noticed a handful of cards I knew had real market value. The owner was likely unaware of the price jump in these cards. The collection had never been systematically catalogued, and manually looking up every card wasn't realistic for them.

I pointed it out and offered to help. The steps were simple, I offered to scan everything, find out what it's actually worth, and give the owner a tool they could use going forward.

Approach

How I Built It

The cards were scanned using Manabox, a mobile app that reads card names and sets via camera and exports a clean CSV. That gave us a structured starting point.

From there, I wrote a Python script to query the Scryfall API for each unique card. Scryfall is the most comprehensive and reliable MTG data source available, offering up-to-date pricing in USD, card variants, and more. The script joined that data back to the original Manabox export, producing an enriched dataset with current market prices per card.

In Power BI, I built a relationship between the original scan data and the Scryfall-enriched table. I used DAX measures to calculate total collection value, value by set, and to flag any card priced above a configurable threshold, to show which ones had no place in the bulk bin.

Dashboard

The Report

The Power BI dashboard gives a view of the full collection: total cards, estimated market value, and a filterable table sorted by price. The owner can filter by set, price range, or condition to quickly identify the cards worth re-pricing or pulling out of the bulk bin.

Interactive Power BI dashboard: filter by set, price range, or condition to explore the collection.
Outcome

What Came Next

The dashboard surfaced a meaningful number of cards that had been sitting in the bulk bin well below market value. The owner now has a complete picture of the collection and a repeatable process: scan with Manabox, run the Python script, refresh the report.

The logical next step I hope to help with for the shop would be migrating this into a lightweight local database, similar to how many card shops run their "single cards for sale" POS.
